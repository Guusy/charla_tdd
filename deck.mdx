<h4 style={{textAlign:"start"}}> TDD es </h4>
![](./src/images/tdd_cycle.png)

<Notes>
Es un herramienta, una manera de desarollar, que esta orientada a los test.
<br/>
1) Test que falle <br/>
2) codigo mas simple para que el test pase (unico objetivo) <br/>
3) (posible refactor) <br/>
4) ver el test pasar ( optional : commit ) <br/>
</Notes>

---
## Ventajas
- Codigo 100% testeado
- No bugs :heart:
- Codigo abierto a refactors totalmente
